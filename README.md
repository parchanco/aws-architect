# Curso AWS - Guía Completa

Este curso está dividido en archivos independientes para facilitar la importación a Notion.

## 📁 Estructura de Archivos

Cada archivo representa una sección del curso y puede importarse individualmente a Notion.

### Archivos Principales
- `00-indice.md` - Índice general y guía de navegación
- `01-introduccion-aws.md` - Conceptos básicos AWS
- `02-iam.md` - Identity and Access Management  
- `03-ec2.md` - Elastic Compute Cloud
- `04-ec2-storage.md` - EBS, EFS, Instance Store
- `05-elb-asg.md` - Load Balancing y Auto Scaling
- `06-rds-aurora.md` - Bases de datos relacionales
- `07-elasticache.md` - Cache en memoria
- `08-route53.md` - DNS y routing
- `09-s3-intro.md` - S3 fundamentos
- `10-s3-advanced.md` - S3 características avanzadas
- `11-s3-security.md` - S3 seguridad
- `12-cloudfront.md` - CDN y aceleración global
- `13-storage-extras.md` - Snow Family, FSx, etc.
- `14-messaging.md` - SQS, SNS, Kinesis
- `15-containers.md` - ECS, Fargate, EKS
- `16-best-practices.md` - Mejores prácticas
- `17-cli-commands.md` - Comandos AWS CLI útiles
- `18-glosario.md` - Términos y definiciones

## 🚀 Cómo Importar a Notion

1. Abre Notion
2. Crea una nueva página para el curso
3. Para cada archivo:
   - Click en "..." → Import
   - Selecciona el archivo .md
   - Notion creará automáticamente la estructura

## 💡 Sugerencias

- **Orden recomendado**: Sigue el orden numérico
- **Para Odoo development**: Archivos 01-07 son esenciales
- **Para producción**: Todos los archivos recomendados
- **Para certificación**: Estudia todos + práctica hands-on

## 📊 Tiempo Estimado

- **Lectura completa**: ~15-20 horas
- **Con práctica**: ~40-50 horas  
- **Dominio completo**: ~3-6 meses (uso regular)

## 🎯 Objetivos de Aprendizaje

Al completar este curso podrás:
- ✅ Desplegar aplicaciones en AWS
- ✅ Configurar infraestructura escalable
- ✅ Implementar alta disponibilidad
- ✅ Gestionar bases de datos en la nube
- ✅ Optimizar costos AWS
- ✅ Aplicar mejores prácticas de seguridad
- ✅ Prepararte para certificaciones AWS

## 📞 Soporte

Este curso está optimizado para desarrollo con Odoo pero los conceptos son aplicables a cualquier aplicación web.
